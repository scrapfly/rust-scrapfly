Monitoring API — aggregated + per-target metrics.

Wraps GET /scrape/monitoring/metrics and GET /scrape/monitoring/metrics/target. Enterprise plan only. See https://scrapfly.io/docs/monitoring#api.

CloudBrowserMonitoringOptions
Options for the Cloud Browser monitoring endpoints. Cloud Browser is session-based (one allocation = one long-lived browser, billed by runtime + bandwidth) and exposes a distinct shape from the request-based products. There is no domain/target and no include_webhook.
