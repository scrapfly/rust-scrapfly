Public schedule client — wraps /scrape/schedules, /screenshot/schedules, /crawl/schedules and the cross-kind /schedules endpoints.

Mirrors the Go and Python SDKs: the kind-specific configuration is supplied to the matching create_*_schedule method; cross-kind list/get/update/ delete/pause/resume/execute work on any schedule by id.
